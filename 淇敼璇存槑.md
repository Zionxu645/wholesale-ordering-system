# V3 修改说明

系统定位由“公开价格网上商城”调整为“服装电子选款册 + 询价管理”。

## 客户前台

- 删除全部价格和金额
- 商品列表显示款号、颜色/尺码数量和供货状态
- 精确库存隐藏
- 浏览和选款无需登录
- 购物车改为选款单，保存在客户浏览器
- 提交选款时登录并生成询价单
- 增加我的询价和正式订单
- 每个商品支持独立链接 `/product/:id`

## 后台

- 新增询价管理
- 询价可标记已联系、已报价、考虑中、未成交
- 谈妥后转为正式订单并扣库存
- 商品支持多图片上传、封面设置和删除
- SKU 只填写颜色、尺码和库存，不填写价格
- 增加朋友圈文案、商品链接和二维码
- 仪表盘删除营收，改为询价、选款件数、客户和低库存

## 数据库

新增：

- `product_images`
- `inquiries`
- `inquiry_items`
- `create_inquiry()`
- `set_inquiry_status()`
- `convert_inquiry_to_order()`
- Supabase Storage `product-images` 桶

旧订单和旧价格字段保留兼容，但前后端不再展示或使用价格。
