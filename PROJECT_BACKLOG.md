# Eluren 项目备忘录

## 暂缓功能：AI 识图自动填写商品资料

状态：暂不开发，等后续有明确需求或合适预算时再接入 API。

目标流程：

1. 在后台“新增商品”窗口上传一张或多张商品图片。
2. 点击“AI 识图填写”。
3. 系统自动识别并预填：
   - 商品名称
   - 款号
   - 分类（短袖、长袖、卫衣、背心）
   - 面料/材质
   - 上新标签
   - 商品描述/卖点
   - 客户可见说明
   - 颜色
   - 尺码
4. 用户检查并手动修改后，再创建商品和 SKU。

必须遵守的识别规则：

- 图片明确出现的信息才自动填写，例如款号、面料、颜色和尺码。
- 无法确定的面料、库存、是否现货等字段必须留空，并提示“需要确认”，不能自行编造。
- 多张图片要综合识别，不只读取第一张。
- AI 只负责预填，最终创建前必须允许人工确认和修改。
- 优先采用最小修改模式，在现有“新增商品”窗口旁增加“AI 识图填写”按钮，不重做整个后台。

可能需要：支持图片理解的模型 API、后端调用接口、结构化 JSON 输出、失败重试和调用成本控制。
