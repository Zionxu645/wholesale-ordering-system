# V3.3 商品运营完整版

本版根据服装批发朋友圈的实际发布方式补齐商品维护功能。

## 商品资料

后台可新增和修改：

- 商品名称
- 对外款号/货号（例如 2689）
- 分类
- 面料/材质（例如 索罗娜凉感极品棉）
- 上新标签（例如 NEW、现货、补货）
- 商品描述/卖点
- 客户可见说明

商品资料会同步到客户选款册和朋友圈素材；历史询价、订单和生产单继续保留提交时快照。

## 颜色、尺码和库存

- 单个 SKU 新增与编辑
- 批量生成颜色 × 尺码组合
- 可选择同步更新已有组合的库存
- 无历史单据引用的错误 SKU 可删除
- 已被询价或订单引用的 SKU 不允许删除，可将库存改为 0

## 商品和图片管理

- 无历史单据引用的商品可在后台删除
- 已有历史单据的商品只能下架，防止破坏记录
- 图片可设为封面、删除、前移和后移
- 商品管理支持按名称、款号、面料和卖点搜索
- SKU 数量较多时默认折叠，页面不再被长列表撑高

## 朋友圈素材

- 简洁版：接近原有朋友圈发布格式，突出款号、面料、颜色、尺码和 NEW 标签
- 详细版：增加商品名称、卖点、客户说明及选款册链接
- 文案生成后可以手动修改再复制
- 保留商品专属链接和二维码

## 客户前台

- 商品卡片显示面料和上新标签
- 商品详情显示面料、标签和客户说明
- 继续隐藏价格和精确库存
